# Tech Stack - Digital Logic Lab

## Frontend
- **Languages:** HTML5, CSS3, Vanilla JavaScript (ES6+).
- **Styling:** Vanilla CSS (no frameworks for initial modules, prioritizing accessibility and performance).
- **Web APIs:** Canvas API, SVG (for diagrams).

## Libraries
- **[html2canvas](https://html2canvas.hertzen.com/):** For rendering diagrams and tables as images for export.
- **[JSZip](https://stuk.github.io/jszip/):** For zipping multiple images (e.g., K-maps) into a single archive.

## Assets
- **Fonts:** [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts.
- **Icons:** Custom SVG or lightweight icon sets.

## Backend & Tooling
- **Grading Script:** Python (for FSM grading and potentially other automated verification tasks).
- **Automation:** GitHub Actions (for deployment and testing).

## Deployment & Hosting
- **Platform:** GitHub Pages.
- **Structure:** Modular structure with each tool hosted under its own sub-path (e.g., `/FiniteStateMachine/`).
