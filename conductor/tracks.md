- [x] **Track: Mux logic tool. This tool is to help students visualize using k-map symmetry along with using muxes. The idea is to generate random 3 variable k-maps (a,b,c) and allow the student to select lines of symmetry in the k-map. Each line of symmetry is the select on a mux. When they choose a line of symmetry, I want the app to help them visualize: "what happens when the select is 0?"/"what happens when the select is 1?" In order to do this I want the portion of the kmap that we are currently interested in to popout slightly and I want the other part of the kmap to darken or fade-out slightly, so that the student only concentrates on the emphasized portion. We may have up to 3 muxes, so the popped out portion may need to also have a line of symmetry selected and again, we need an emphasized section and a de-emphasized section from this already smaller section. As we emphasize each section, I also want to have a mux (or several muxes) at the bottom of the screen that highlights what portion (according to our kmap) that we are currently working on. The user should have the option of difficulty (1,2,or 3 mux problems). Each time that we create a line of symmetry, that line should show up in a bright color that is color coded with the select on the mux diagram. also, each line of symmetry should be a wavy line when it is drawn on the kmap.**
